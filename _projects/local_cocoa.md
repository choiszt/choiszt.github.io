Cocoa enables users to build a private, on-device knowledge system without cloud dependencies. The application combines file ingestion, intelligent chunking, and vector-based retrieval to create a persistent memory space.

### Desktop Version — [Local Cocoa](https://github.com/synvo-ai/local-cocoa)
- Fully local inference with zero data transmission
- Multimodal memory supporting documents, images, audio, and video
- Vector-powered semantic search using local Qdrant
- Hardware-accelerated processing optimized for Apple Silicon and consumer GPUs
- Integrated note-taking that becomes part of searchable memory

### Mobile Version — [Mobile Cocoa](https://github.com/synvo-ai/mobile-cocoa)
- Mobile-first AI coding assistant built with React Native (Expo)
- AI-assisted coding discussions and design directly from your phone
- Concurrent sessions supporting multiple parallel development sessions
- Docker integration for sandboxed command execution
- E2E encryption with optional AES-256-GCM for secure access
