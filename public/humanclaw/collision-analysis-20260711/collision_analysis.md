# HCBv6 碰撞说明书：benchmark 里的 collision 到底是什么（2026-07-11 v4）

> 纯客观描述，配视频示意（网页版：https://choiszt.com/humanclaw/collision-analysis-20260711/）。
> 数据 = 16 模型 × ~1,188 集步级接触记录（约 136 万步），剔除下陷场景与 init-stuck；论文口径 9 个入表模型。
> 目标判定 = 被撞物语义类别与任务类别匹配（ObjectNav 任意实例）。
> v4 修订：目标判定改语义类别匹配（此前用目标 AABB 包络盒，对多实例类别把"目标接触"高估至 ~50%、把地毯低估至 12–19%）。

## 0. 一页结论

1. **碰撞的主体是"移动中与环境家具的接触"**：推着/蹭着物体走 26.7–43.4%（最大类）+ 路过擦碰 18.5–38.2%，合计约 2/3。**真正碰到目标类别的只占 4.0–19.7%**，模型分化大——GPT 19.7%（贴着目标耗步）vs Claude 4.5%（到不了目标）。
2. **地毯是第二大接触来源**：20–33% 的接触步只踩了地毯/垫子（测量噪声）；剔除后 Coll% −9~−14 pt，排序不变。
3. **碰撞发生那一刻约 92% 的接触点不在当前 ego 画面里**（行走步池化）：侧面 42.1% + 身后 37.5% + 正前但画面下缘以下 12.5%；画面内正前方仅 7.9%（16 模型全在 7–12%）。
4. **下肢主导**：腿 42–55% ≫ 手臂 24–37% ＞ 躯干 ＞ 头 4–9%。
5. **70% 接触步发生在转向/侧移/静止中**（视角扫动），仅 30% 在直线行走中。
6. **模型收不到任何碰撞信号**（系统事实）：环境每步计算反馈字符串（`HalfPhysicsEnv.py:777`），但 planner/verifier prompt 构造均显式丢弃（`psv_plan_skill_planner.py:347` / `verifiers/v0.py:178` 的 `del env_feedback`）；模型唯一感知通道是 ego 图像。

## 1. 定义与口径

- 计数规则（与论文 Coll% 一致）：一步内任一身体 link 接触非地面物体即记碰撞步；"地面"= stage（含墙体、地板）——**墙体接触不计入**。
- 事件化：连续接触步合并为一个事件，按持续步数加权。
- 采样：每步取最后一物理帧（绝对率略低估、构成无偏）；约 25% 对象语义未解析（unknown，疑似门等构件）。

## 3. 碰在身体哪里

腿 42–55% ≫ 手臂 24–37% ＞ 躯干 13–22% ＞ 头 4–9%（全模型一致）。

## 4. 方位与画面可见性

只对行走锚点步判定（朝向=位移方向）；ego 相机 448×448、HFOV 90°、下俯 23°、高≈骨盆+0.7m。**9 模型池化（非目标接触点）**：画面内正前 7.9% / 正前但画面下缘以下 12.5% / 侧面 42.1% / 身后 37.5%。判定的是接触点是否在当前帧视锥内（画面外 ≠ 从未见过；画面内也可能被遮挡）。约 70% 的接触步在转向/侧移/静止中（视角扫动，不参与此分布）。

## 6. 类内小花样

- **连环碰撞**：≥2 类物体的事件占 17.8%（≥3 类 2.4%），承载 42.5% 碰撞步；高发于顶着不动/推着走；高频组合 chair+table、couch+table。样例：Gemini-3.1 / 102344094 / ep20 / 21–45 步（cabinet→bathtub）。
- **头部撞击**：占步数 1.4–3.6%（Claude 3.6、InternVL 3.2 最高）；对象 top：wardrobe、mirror、window、picture、**灯具**——头撞灯不罕见：GPT 263 步、InternVL 255、Claude 249。样例：Gemini-3.1 / 102344094 / ep6 / step7（table_lamp 怼脸）。
- **坐到错误物体**：有 sit 的 episode 中 18–40% 出现 sit 时髋/骨盆接触非目标（Claude 40.3%、Gemma 36.9%、Gemini-3.1 34.9%、GPT 32.8%）；坐上 chair/cabinet/bathtub/shower。样例：Gemini-3.1 / 102344094 / ep20 / 24–48 步（压上 cabinet）。
- **上下台阶磕碰**：climb 步少（55–1,215/模型）但 45–69% 的 climb 步在碰东西。样例：Gemini-3.1 / 102344094 / ep17 / 10–17 步。
